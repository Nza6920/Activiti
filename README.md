Activiti
========

> ## 升级 activiti-spring-boot
> 1. cd \modules\activiti-spring-boot 
> 2. mvn clean install source:jar -Dmaven.test.skip=true


> ## 升级 activiti-ui 
> 1. cd \modules\activiti-ui
> 2. mvn clean install source:jar -Dmaven.test.skip=true

> ## 测试访问
> 1. cd \modules\activiti-ui\activiti-app
> 2. mvn clean tomcat7:run
